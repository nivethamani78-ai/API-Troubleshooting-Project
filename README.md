API & SQL Troubleshooting Project

This project simulates real-time L2 Technical Support scenarios using API and SQL troubleshooting techniques.

Tools Used

- Postman
- SQL (MySQL / SQLite)

Project Overview

This project includes multiple troubleshooting scenarios commonly faced by Technical Support Engineers.

Topics Covered

- API errors (400, 401, 403, 404, 500)
- Request/Response debugging
- Database issues (missing data, duplicate data, NULL values)
- Root cause analysis

Screenshots

Screenshots of API errors and successful responses are included in the repository.

Tickets

- Created 15 real-world troubleshooting tickets
- Each ticket includes Issue, Root Cause, and Solution

Objective

To demonstrate practical knowledge in API, SQL, and troubleshooting similar to L2 Technical Support roles.
